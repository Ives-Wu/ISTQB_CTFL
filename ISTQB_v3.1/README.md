# **ISTQB 基礎級軟體測試參考書**

## **版權說明**
本參考書是針對ISTQB初級大綱2018版本而編寫的。ISTQB大綱版權歸ISTQB所有，可以在ISTQB官方網站下載。
本參考書旨在幫助大家更好的學習ISTQB初級大綱的內容，在尊重和註明來源的情況下，可以自由的使用和傳播。
本參考書編寫過程中，參考了大量的資料。文中盡可能的將所有參考的資料的來源做了標註。但是仍然可能存在一些遺漏的地方，如果對此有異議，可以及時聯系我們進行更新。
